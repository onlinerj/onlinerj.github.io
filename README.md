# Rajat Jaiswal's Personal Website

This site is built with [Jekyll](https://jekyllrb.com/) using the [Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) theme.

## About

Personal website of Rajat Jaiswal - AI Research Fellow & Generative AI Lead

## Quick Start

### Prerequisites

- Ruby 3.0+
- Bundler

### Installation

```bash
bundle install
```

### Running Locally

```bash
bundle exec jekyll serve
```

Visit `http://localhost:4000` in your browser.

## Adding Content

### Blog Posts

Create a new file in `_posts/` with the format:

```
YYYY-MM-DD-title.md
```

Example front matter:

```yaml
---
title: My Post Title
date: 2024-01-01 12:00:00 -0500
categories: [Category1, Category2]
tags: [tag1, tag2]
---

Your content here...
```

### Pages

Add or edit files in `_tabs/` folder to modify navigation tabs.

## Configuration

Edit `_config.yml` to customize site settings.
