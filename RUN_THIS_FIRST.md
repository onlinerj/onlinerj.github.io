# 🚀 RUN THIS FIRST!

## Your site has been migrated to Chirpy theme!

Follow these 3 simple steps to get started:

---

## Step 1: Install Dependencies

### On Windows:
```cmd
setup.bat
```

### On Mac/Linux:
```bash
chmod +x setup.sh
./setup.sh
```

### Or manually:
```bash
bundle install
```

---

## Step 2: Start the Development Server

```bash
bundle exec jekyll serve
```

Then open your browser and visit:
```
http://localhost:4000
```

---

## Step 3: Review Your Site

Check these pages:
- ✅ Home page (blog posts)
- ✅ About page
- ✅ CV page
- ✅ Archives
- ✅ Categories
- ✅ Tags

---

## 🎉 That's it!

Your site is now running with the Chirpy theme.

2. **Customize your content:**
   - Edit `_tabs/cv.md` with your actual CV
   - Add more blog posts in `_posts/`

3. **Deploy to GitHub:**
   ```bash
   git add .
   git commit -m "Migrate to Chirpy theme"
   git push
   ```


### Ruby not installed?
